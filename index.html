CSS Animation Cheat Sheet

close
.box
Usage transition

          .box {
            width: 200px;
            height: 200px;
            background: #F4989C
            transition-property: border-radius;
            transition-duration: 2s
            border-radius: 50%;
          }
Usage animation

          .box {
            width: 200px;
            height: 200px;
            background: #F4989C;
            border-radius: 5%;
            animation-name: box-animation;
            animation-duration: 2s;
            animation-iteration-count: infinite;
          }

          @keyframes box-animation {
            0% {
              transform: none;
            }
            100% {
              border-radius: 50%;
            }
          }
